## 1. 这是什么

一套用 Python 做的 **LFP（磷酸铁锂）电池回收经济与环境分析**。

输入：1万吨/年废旧 LFP 电池  
输出：三种回收方法各赚多少钱、排多少碳  
验证：对比了 Argonne 国家实验室官方文档和两篇顶刊论文

**三种回收方法：**

| 方法 | 英文 | 原理 | 一句话 |
|------|------|------|--------|
| 火法 | Pyrometallurgical | 高温熔炼（1500°C） | 扔进炉子里烧，金属熔化分层 |
| 湿法 | Hydrometallurgical | 化学试剂浸泡溶解 | 用酸/碱把金属溶解再分别提取 |
| 直接回收 | Direct Recycling | 物理分离+修复 | 拆开电池，把正极材料直接拿出来复用 |

## 2. 背景知识：5分钟看懂电池回收

### 2.1 LFP 电池里有什么

一颗 LFP 电池电芯 = 一堆材料按比例组成：

```
正极材料（LFP 粉末）: 28%   ← 最值钱的东西
铝箔（集流体）:      20%   ← 可以卖废铝
石墨（负极）:        16%   ← 可以回收
铜箔（集流体）:      15%   ← 最值钱的金属
钢壳:                8%
电解液:              5%
粘合剂+导电剂+塑料:   8%
```

### 2.2 回收是干什么

废旧电池 → 拆解+处理 → 把里面的材料分离开 → 卖掉

赚什么钱 = 卖材料收入 + 收"进门费"（电池厂付钱请你处理）  
花什么钱 = 设备折旧 + 人工 + 电费/天然气 + 化学试剂 + 废水处理

### 2.3 为什么三种方法结果差这么多

**火法（Pyro）：**
- 把电池扔进 1500°C 炉子烧
- 锂和铝会混进渣里，**拿不回来**
- LFP 没有钴/镍这些贵金属，所以火法回收 LFP = 亏本
- 只适合含钴的电池（NMC/NCA）

**湿法（Hydro）：**
- 用硫酸+双氧水把金属溶解，再一步步提取
- 可以回收锂（做成碳酸锂 Li₂CO₃ 卖钱）
- 但要用大量化学品，成本不低
- LFP 的锂含量 4.4%，碳酸锂价格是关键变量

**直接回收（Direct）：**
- 物理方法拆开电池
- 把 LFP 正极粉末直接拿出来，稍微修复就能用
- **不用重新合成正极材料**，省了最大的一块成本
- LFP 正极值 ~$10-14/kg，直接回收最划算

## 4. 怎么读取 Excel 模型

### 4.1 问题：openpyxl 打不开

EverBatt 的 xlsm 文件比较老，里面有些 XML 数据超出了 openpyxl 的校验范围。

**解决方案（完整代码在 `read_everbatt.py`）：**

```python
# 思路：xlsm 本质是个 zip 包，解压后修 XML，再打包
import zipfile, os, tempfile
from xml.etree import ElementTree as ET

# 1. 解压 xlsm
with zipfile.ZipFile('EverBatt 2023.xlsm', 'r') as z:
    z.extractall(temp_dir)

# 2. 修复所有 XML 中的非法值（字体 family > 14、主题色 > 10）
for xml_file in find_all_xml_files(temp_dir):
    tree = ET.parse(xml_file)
    for el in tree.iter():
        if el.tag.endswith('family') and int(el.get('val', 0)) > 14:
            el.set('val', '2')  # 改成合法值
        if 'theme' in el.attrib and int(el.get('theme', 0)) > 10:
            el.set('theme', '0')
    tree.write(xml_file)

# 3. 重新打包
with zipfile.ZipFile('everbatt_fixed.xlsm', 'w') as z:
    for f in files:
        z.write(f, arcname)

# 4. 现在可以正常用 openpyxl 打开了
wb = openpyxl.load_workbook('everbatt_fixed.xlsm', data_only=True)
print(wb.sheetnames)  # 17 个工作表
```

## 5. 分析方法：成本/收入/利润/GHG 怎么算

### 5.1 整体思路

```
输入：1万吨 LFP 电池 × 材料占比(%) = 每种材料有多少吨

对于每种回收方法：
  ├── 收入 = Σ(回收到的材料 × 市场价) + 进门费收入
  ├── 成本 = 化学品 + 能源 + 人工 + 设备折旧 + 维护 + 管理费 + 税费
  ├── 利润 = 收入 - 成本
  └── GHG排放 = 能源排放 + 化学品排放 + 焚烧排放 - 避免的 virgin 生产排放
```

### 5.2 收入计算（Revenue）

```python
# 伪代码
收入 = 0

# 1. 进门费（gate fee）
# EverBatt Table 13: LFP 的 fee = -$2/kg
# 负号 = 电池厂付钱给回收商
收入 += -fee × 总重量  # -(-$2) × 10,000吨 = +$20M

# 2. 卖材料
for 每种材料 in 电池成分:
    回收量 = 材料含量 × 回收率(方法)
    收入 += 回收量 × 市场价

# 例：Direct 回收 LFP 正极
# 回收量 = 2800吨 × 90% = 2520吨
# 收入 = 2520吨 × $14,000/吨 = $35.28M
```

### 5.3 成本计算（Cost）

用的是 **化工厂标准成本模型**（EverBatt Table 15）：

```
1. 直接成本
   ├── 化学品：Σ(每kg电池消耗的化学品 × 化学品单价)
   ├── 能源：电费 + 天然气费
   ├── 人工：工人工资（35人 × 3班倒）
   └── 废物处理：没回收的材料进填埋场

2. 固定成本（CapEx 分摊）
   ├── 设备折旧：设备总投资 ÷ 10年
   ├── 维护：设备投资的 6%
   ├── 税费：固定资本 4%
   ├── 保险：固定资本 1%
   └── 利息：总资本 5%

3. 管理费
   ├── 工厂管理费：50% × (人工 + 维护)
   └── GSA（行政+销售+研发）：~15%总成本
```

### 5.4 利润计算（Profit）

```python
利润 = 收入 - 成本
# > 0 → 赚钱
# < 0 → 亏本
```

### 5.5 GHG 排放计算

```
排放 = 能源排放 + 化学品生产排放 + 焚烧排放 - 避免的 virgin 材料排放

# 能源排放：用电/天然气产生 CO₂
# 化学品排放：生产硫酸/烧碱/双氧水过程排放
# 焚烧排放：火法把石墨/塑料/电解液烧掉 → CO₂
# 避免排放：回收的金属/材料不用从矿石重新炼 → 省下的排放（负值）
```

### 5.6 完整代码在哪

文件 `lfp_analysis_verified.py`，约 350 行。

结构：
```
第 1-90 行：参数定义（成分/价格/回收率/化学品/能源/成本模型）
第 91-200 行：analyze_method() 函数（收入/成本/利润/GHG 计算）
第 201-290 行：运行三种方法 + 打印结果
第 291-350 行：画 6 张对比图
```

## 6. 怎么验证结果是否正确

### 6.1 对照什么文献

| 文献 | 发表期刊 | 内容 |
|------|----------|------|
| **ANL-19/16** (2019) | Argonne 官方文档 | EverBatt 完整方法论，88页 |
| **Xu et al. (2020)** | Joule (IF 39) | 用 EverBatt 2019 分析直接回收 |
| **Ji et al. (2024)** | Nature Communications | 用 EverBatt 对比三种回收路线 |

### 6.2 验证步骤

**第 1 步：检查参数来源**
- 每个数字都要能追溯到 Argonne 文档的哪个表
- 比如「LFP 正极直接回收率 = 90%」来自 Table 12

**第 2 步：和文献对比成本**
```
我的 Pyro 成本 vs 文献：
  $2.31/kg  vs  Xu $3.4/kg  vs  Ji $3.0-3.5/kg

有点低 → 说明我的 CapEx 或能耗估计偏乐观了
但量级正确（都是 $2-4/kg 范围）
```

**第 3 步：对比定性结论**
```
文献结论： Direct >> Hydro > Pyro（对 LFP）
我的结论： Direct >> Hydro > Pyro  ✓ 一致
```

**第 4 步：敏感性分析**
- 改动关键参数（碳酸锂价格、进门费、电价）
- 看利润方向是否和文献一致

## 9. 常见问题

### Q1: 为什么不直接用 Excel 跑 EverBatt？

A: EverBatt 是 .xlsm 文件（带 VBA 宏），需要启用宏的 Excel。而且参数非常多（100+ 个），直接在 Excel 里一个个改很麻烦。用 Python 做的好处：
- 参数全在代码顶部，一目了然
- 改一个数字重跑只需 1 秒
- 自动画图、自动生成对比表
- 可以批量做敏感性分析

### Q2: 这个分析准不准？

A: 
- **定性结论准**：三种方法的排名（Direct > Hydro > Pyro）在所有文献中一致
- **定量有误差**：成本取决于具体工厂的规模、地点、自动化程度，文献之间的数字也不完全一样（Pyro 成本 $2.5-3.5/kg 都是合理的）
- **趋势可靠**：用于对比"哪种方法更好"足够准确；用于写商业计划书需要更详细的工程数据

### Q3: 为什么 Pyro 的利润有争议？

A: LFP 的进门费是最关键的变量：
- 2019 年：$2/kg（锂便宜，没人要 LFP 废电池）
- 2022 年：~$0（碳酸锂涨到 $70/kg，抢着回收）
- 2023 年：$0-0.5（碳酸锂回落到 $17/kg）

这就是为什么同一套方法、同样的论文，换个年份结论就不同。

### Q4: 怎么深入学习？

推荐阅读顺序：
1. 先看懂 `lfp_analysis_verified.py` 的参数定义部分（第 1-90 行）
2. 再看 `analyze_method()` 函数（收入→成本→利润→GHG，四步清晰）
3. 读 Argonne 文档（`EverBatt 2019 Documentation.pdf`）的关键表格（Table 10-15）
4. 搜索 "EverBatt LFP" 看最新的论文
